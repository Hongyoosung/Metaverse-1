# aries-python-test

Hyperledger Aries Cloud Agent Python 코드 테스트를 위해 작성된 저장소

### 실행방법

윈도우 환경에서 실행하였으며 python 3.11.3 버전 사용

        $python --version
        Python 3.11.3

pip를 사용해 패키지 다운

        $pip --version
        pip 23.1

패키지의 경우 aries-python-test 폴더로 이동 후 pip로 패키지 다운

        $pip install -r requirements.txt --user

python을 사용해 코드 실행

        $python main.py
